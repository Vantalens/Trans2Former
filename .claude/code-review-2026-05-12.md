## 发现的问题

### 🔴 严重问题

#### 1. PDF 高保真输出坐标计算错误 (已修复)

**位置**: `public/formats/pdf-output-high-fidelity.js:75`

**问题描述**:
```javascript
// 错误代码
const dx = x - (run.bbox.x || 0);  // 始终为 0
```

计算相对偏移量时错误地使用了 `run.bbox.x - run.bbox.x`，导致同一行内的文本定位不正确。

**修复方案**:
```javascript
// 修复后
let lastX = null;
// ...
const dx = x - lastX;  // 正确计算相对偏移
lastX = x;
```

**影响**: 高保真 PDF 输出中同一行的多个文本片段会重叠显示。

**状态**: ✅ 已修复并验证
