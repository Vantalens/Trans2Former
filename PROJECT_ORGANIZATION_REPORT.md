## 7. 待改进事项

### 7.1 高优先级 (P1)

1. **提升分支覆盖率**
   - 当前: 71.95%
   - 目标: 80%+
   - 方案: 添加错误路径和边界条件测试

2. **添加代理对支持**
   - 问题: PDF ToUnicode CMap 不支持 UTF-16 代理对
   - 影响: CJK Extension B、Emoji 等字符
   - 方案: 扩展 codespace 为 3 个范围

3. **添加 Node.js 版本约束**
   - 问题: puppeteer 要求 Node.js >= 22.12.0
   - 方案: 在 package.json 添加 engines 字段
